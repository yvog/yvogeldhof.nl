Quick tip of the day... let VS Code fix ESLint errors automatically when the document is saved.
Add this to your `settings.json` file, found in the `.vscode` directory in your project.

## Prerequisite

- [ESLint extension for VS Code](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

## Snippet

```javascript
{
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "eslint.validate": ["javascript", "typescript"],
}
```

Never fix those 'unnecessary curly braces around strings' errors manually again!
