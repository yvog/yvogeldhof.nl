Everyone is setting up their VS Code editor to their personal preferred settings. As a result, we are all using different VS Code extensions as well.
This post might help anyone looking for interesting VS Code extensions. These are extensions I use in 2023:

- [Color Highlight](https://marketplace.visualstudio.com/items?itemName=naumovs.color-highlight)
  - Makes colors in the document visible directly in the editor.
- [Material Theme](https://marketplace.visualstudio.com/items?itemName=Equinusocio.vsc-material-theme) + [Material Theme Icons](https://marketplace.visualstudio.com/items?itemName=Equinusocio.vsc-material-theme-icons)
  - My spaghetti code will be disguised in beautiful colors using this extension.
- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)
  - Integrates ESLint JavaScript into VS Code.
- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
  - Code formatting using Prettier.
- [Git Blame](https://marketplace.visualstudio.com/items?itemName=waderyan.gitblame)
  - Who wrote this line?!
- [MDX](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx)
  - Adds language support for [MDX](https://github.com/mdx-js/mdx)
- [MDX Preview](https://marketplace.visualstudio.com/items?itemName=xyc.vscode-mdx-preview)
  - Lets you preview MDX easily in split view.
- [Reveal](https://marketplace.visualstudio.com/items?itemName=smulyono.reveal)
  - Inspired by JetBrains PHPStorm, this extension brings a button to the editor which highlights the current file in the sidebar when clicked.
- [YAML](https://marketplace.visualstudio.com/items?itemName=redhat.vscode-yaml)
  - YAML language support.
- [XML](https://marketplace.visualstudio.com/items?itemName=redhat.vscode-xml)
  - XML language support.
- [JavaScript Booster](https://marketplace.visualstudio.com/items?itemName=sburg.vscode-javascript-booster)
  - Brings refactoring commands, like 'sort imports' to the editor.
- [GraphQL](https://marketplace.visualstudio.com/items?itemName=orsenkucher.vscode-graphql)
  - Adds GraphQL syntax highlighting, validation, and language features like 'go to definition' and more to the editor.
- [Azure Resources](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azureresourcegroups)
  - View and manage Azure resources.
- [Azure Functions](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azurefunctions)
  - Work with Azure functions in VS Code.
- [Azure Account](https://marketplace.visualstudio.com/items?itemName=ms-vscode.azure-account)
  - Azure Sign-In and Subscription management tool for VS Code.
